# KevScores-Public
Fantasy Baseball Rankings / Trade Calculator

Fantasy football trade calculators are all over the internet, why not baseball?

I've went ahead and created a multi-faceted baseball analysis tool, but not the usual deep dive analytics you're used to seeing, just a simple value calculator.

How do you value good players? Great players? The goal of this project is to solve the relative value of baseball players, accounting for hot streaks, day-to-day fluctuations, and other changes in value that ranking leaderboards are just not tracking.

The "Kev Score" metric combines ESPN's "Value" data for a player with how they've scored so far this year. This creates a moving value with a "baseline" that takes the player's talent into account.

I've also created two versions of the calculator, Kev Score and "Weighted" score. This helps for those trades your leaguemates send where it's 5 bad players for one good player.

Thank you all for reading, and enjoy!
